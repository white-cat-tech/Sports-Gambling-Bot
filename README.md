# Sports-Gambling-Bot
Gave myself a month to see if I can create a model that makes money betting on sports. This is the result.

## Installation

Install the relevant requirements with:

```bash
pip3 install -r requirements.txt
```

## Usage

Run the core program with

```bash
python model.py
```
